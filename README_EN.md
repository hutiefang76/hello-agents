## 🎯 Project Introduction

&emsp;&emsp;If 2024 was the year of the "Battle of a Hundred Models," then 2025 has undoubtedly ushered in the "Year of Agents." The focus of technology is shifting from training larger foundation models to building smarter agent applications. However, systematic, practice-oriented tutorials are extremely scarce. This repository compiles such material from public study resources, providing a comprehensive guide to building agent systems from scratch.

&emsp;&emsp;Today, agent development is mainly divided into two schools: one is software engineering-oriented agents like Dify, Coze, and n8n, which are essentially process-driven software development with LLMs serving as data processing backends; the other is AI-native agents, truly AI-driven agents. This material aims to lead you to deeply understand and build the latter — truly AI Native Agents — starting from core principles, delving into core architecture, understanding classic paradigms, and ultimately building your own multi-agent applications.

## 💡 How to Learn

&emsp;&emsp;Welcome, future builder of intelligent systems! This material balances theory and practice, helping you systematically master the entire process of designing and developing from single agents to multi-agent systems. It is suitable for **AI developers, software engineers, students** with some programming foundation, and **self-learners** with a strong interest in AI. You should have basic Python skills and conceptual understanding of LLMs (e.g., calling an LLM via API). The focus is on application and construction.

&emsp;&emsp;The material is divided into five parts:

- **Part 1: Fundamentals** (Chapters 1-3) — agent definitions, types, history, and LLM core knowledge.
- **Part 2: Building Your Agent** (Chapters 4-7) — ReAct paradigms, low-code platforms, frameworks, and building from scratch.
- **Part 3: Advanced** (Chapters 8-12) — memory, context engineering, training, communication protocols, evaluation.
- **Part 4: Cases** (Chapters 13-15) — travel assistant, deep research agent, cyber town.
- **Part 5: Capstone** (Chapter 16) — your own complete multi-agent application.

&emsp;&emsp;Agents are a rapidly developing field that is extremely dependent on practice. The `code` folder contains all supporting code. **Combine theory with practice** — run, debug, and modify every piece of code.
